Reflash manual do cartao /dev/sdb para Raspberry Pi com acesso SSH

1. Desmontar as particoes montadas:
sudo umount /dev/sdb1
sudo umount /dev/sdb2

2. Gravar a imagem Bookworm no cartao:
xzcat /tmp/opencode/raspios-bookworm-arm64.img.xz | sudo dd of=/dev/sdb bs=4M conv=fsync status=progress

3. Garantir flush final:
sync

4. Retirar e voltar a inserir o cartao.

5. Depois de reinserir, o sistema deve montar a particao FAT de boot. Se nao montar, voltar a inserir.

6. Activar SSH na particao boot:
touch /media/rjpg/bootfs/ssh

7. Criar utilizador inicial (substituir USERNAME e PASSWORD):
echo 'USERNAME:$(echo "PASSWORD" | openssl passwd -6 -stdin)' > /media/rjpg/bootfs/userconf.txt

8. Opcional: definir hostname mais tarde ja no Raspberry como photo-plant.

9. Arrancar o Raspberry so com alimentacao + Ethernet + cartao.
